# Keyboard Shortcuts Specification

Power user shortcuts speed up simulation design and debugging workflows.

## 1. Global Navigation Shortcuts

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `g` then `s` | Go to Simulations | Navigates the router path directly to `/` |
| `g` then `t` | Go to Topologies | Navigates the router path directly to `/topologies` |
| `g` then `w` | Go to Workers | Navigates the router path directly to `/workers` |

## 2. Workspace Control Shortcuts

| Shortcut | Action | Scope |
| :--- | :--- | :--- |
| `Spacebar` | Play / Pause Simulation | Active Simulation Detail route |
| `Escape` | Close active Inspector Drawer / Dialog | Anywhere |
| `Ctrl + Z` | Undo last connection line link | Topology Canvas Editor |
| `v` | Switch to Select tool | Topology Canvas Editor |
| `h` | Switch to Hand / Pan canvas tool | Topology Canvas Editor |
