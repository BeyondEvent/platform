# @beyondevent/ui

Shared React component library for BeyondEvent. No application state.

## Exports

- `Button` — primary/secondary/ghost/destructive variants
- `Card`, `CardHeader`, `CardTitle`, `CardContent`
- `Badge` — default/success/warning/destructive/outline variants
- `LoadingSpinner`
- `ErrorBoundary`

## Peer Dependencies

- `react@^19`
- `react-dom@^19`
