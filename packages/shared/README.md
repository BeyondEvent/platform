# @beyondevent/shared

Shared types, utilities, errors, and constants used across all BeyondEvent packages.

No runtime dependencies.

## Exports

- **Types**: `EventId`, `WorkerId`, `SimulationId`, `TopologyId`, `Timestamp`, `SemVer` (branded strings)
- **Errors**: `BeyondEventError`, `ConfigurationError`, `ValidationError`, `NotFoundError`
- **Utils**: `generateId()`, `assertNever()`, `isNonNullable()`
- **Constants**: `APP_NAME`, `DEFAULT_TIMEOUT_MS`, `MAX_RETRY_ATTEMPTS`, `DEFAULT_BATCH_SIZE`
