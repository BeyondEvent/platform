# @beyondevent/topology

Directed graph for BeyondEvent topologies. No UI code.

## Exports

- **Types**: `NodeId`, `EdgeId`, `TopologyNode`, `TopologyEdge`, `ValidationResult`, `TopologySnapshot`
- **Interfaces**: `ITopologyGraph`
- **Factory**: `createTopologyGraph()` — in-memory directed graph with BFS traversal and validation
