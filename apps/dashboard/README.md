# @beyondevent/dashboard

Visual interface for BeyondEvent.

## Features (Phase 4+)

- React Flow topology canvas (drag-and-drop design)
- Real-time event flow visualization
- Worker inspector
- Queue inspector
- Metrics dashboard
- Trace timeline

## Development

```sh
pnpm dev
```

Dashboard starts on `http://localhost:5173`. Proxies `/api` and `/socket.io` to the API on port 3000.
